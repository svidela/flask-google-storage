Welcome to Flask-GoogleStorage
==============================

**Flask-GoogleStorage** is a Flask extension for adding storage capabilities using Google Cloud Storage.

Install
=======

Flask-GoogleStorage requires Python >= 3.6.

.. code-block:: bash

    $ pip install flask-googlestorage
